How It Works

A Radically Simple Code Font Workflow

We've removed all the friction from code font management. Get up and running in three simple steps.

1

Install Extension

Find and install "CodeFont" from the Visual Studio Code Marketplace in seconds.

2

Open Font Explorer

Click the newly added CodeFont icon in your Activity Bar to see all your available code fonts, intelligently filtered.

3

Click to Apply

Simply click on any code font name in the list to instantly apply it to your editor. That's it! Enjoy.
